- Clone the repository
```
git clone  https://github.com/devtoolsweb/ts-goodies.git <project_name>
```
- Install dependencies
```
cd <project_name>
npm install
```
- Build and run the project
```
npm start
```
## Testing
The tests are written in Jest

### Running tests using NPM Scripts
````
npm run test

````
